# Оно тебе надо

## Описание проекта
Веб-сайт аукциона "Оно тебе надо" - учебный проект по курсу Full-stack разработки на Яндекс.Практикуме.

## Технологии
- HTML5
- CSS3
- Flexbox
- Grid Layout

## Функциональность
Сайт включает следующие разделы:
- Главная страница с каталогом лотов
- Информация об аукционе
- Форма для размещения лотов

## Запуск проекта
Откройте файл `index.html` в браузере.

## Статус
Проект закончен.
